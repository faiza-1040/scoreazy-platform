# Confidence Building Microcourse - Scoreazy

## Project Overview

This project is a web application designed for **Scoreazy**, offering a **Confidence Building Microcourse** for **Grades 3–5**. It empowers children to **speak up, participate, and grow** in just one week.

The course uses proven methods from educational psychology, helping students build confidence through **fun games**, **guided stories**, and **reflection activities**. 

The web page includes sections like:
- **Hero Section**: Introduction to the course.
- **About Us**: Describes the course and its benefits.
- **Statistics**: Highlights the reach and success of the program.
- **FAQ Section**: Answers common questions.
- **CTA (Call to Action)**: Encourages users to enroll.
- **Testimonials**: Showcasing positive feedback from parents.
